## 🧠 Problem

Writing technical blogs is time-consuming and requires structured thinking.

Developers struggle with:

* Organizing content
* Writing clear explanations
* Maintaining consistency

---

## 💡 Solution

Dev-AI-Blog integrates AI into the writing workflow.

Users can:

* ✨ Generate structured blog content
* 🧠 Improve technical writing
* ⚡ Reduce content creation time
* 📝 Manage blogs efficiently

---

## ⚙️ Tech Stack

| Layer      | Technology   |
| ---------- | ------------ |
| Frontend   | React.js     |
| Styling    | Tailwind CSS |
| State Mgmt | Context API  |
| API Calls  | Axios        |
| Hosting    | Vercel       |

## 🔥 Features

* 🔐 Authentication UI
* ✍️ AI Blog Generation Interface
* 📝 Blog Create / Edit / Delete
* ⚡ Responsive UI
* 🔗 API integration with backend

---

## 🧪 Workflow

```text
User → Login/Register  
      → Enter Topic  
      → AI Generates Blog  
      → Edit → Save  
```

---

## 🏗️ Installation

```bash
git clone https://github.com/AEVILOP/Dev-AI-Blog.git
cd Dev-AI-Blog
npm install
```

---

## 🔑 Environment Variables

```env
VITE_API_BASE_URL=https://dev-ai-blog-backend.onrender.com
```

---

## ▶️ Run Locally

```bash
npm run dev
```
